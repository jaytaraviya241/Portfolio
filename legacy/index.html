Featured Case Study

A Fulfillment System
Built Without Apps

Pie Business — eCommerce Fulfillment

Custom Pie Delivery & Pickup System on Dawn Theme

🧩 Problem

Client needed complex fulfillment logic — zip code gating, bundle building, conditional pickup vs. delivery UX — without third-party apps that would slow the store and add recurring costs.

⚙️ Solution

Built a PIE_CONFIG global metafield system as single source of truth, a PieRules.evaluate() rules engine, AJAX Cart API integration, and a cart_checkout_validation Shopify Function for server-side enforcement.

📊 Result

Zero third-party app dependencies. Merchant-editable business rules via metafields. 97 Lighthouse performance score on live store.
